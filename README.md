
# AI-Powered Customer Churn Analytics System

## Project Overview
This project predicts customer churn using Machine Learning and visualizes business insights using Power BI.

## Tools & Technologies
- Python
- Pandas
- Scikit-learn
- SQL
- Power BI
- GitHub

## Business Problem
Telecom companies lose customers due to poor retention strategies.
This system identifies high-risk customers and helps management reduce churn.

## Features
- Customer churn prediction model
- SQL business analysis
- Power BI dashboard
- Revenue analysis
- Churn risk segmentation
